# Website Creator Marketplace

A Claude Code plugin marketplace for creating WordPress websites.

## Installation

```bash
/plugin marketplace add burtrw/website-creator
```

Then install the plugin:

```bash
/plugin install website-creator
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [website-creator](./plugins/website-creator) | Create WordPress block themes from simple descriptions with live Playground preview |
